# KayPolls Option Images

Place local poll option images here. Vite serves this folder from `/nominees/...`.

Current mapped folders:
- `food/`: pizza, burger, fries, shawarma
- `colors/`: blue, black, red, purple
- `pet/`: dog, cat, bunny, parrot
- `drink/`: coke, fanta, tea, smoothie
- `social_media/`: tiktok, instagram, x, snapchat

The frontend mapping lives in `frontend/src/config/voteOptions.js`.
